# Floating Contact Button

A customizable floating contact button with multiple links (WhatsApp, Facebook, LinkedIn, etc.) for WordPress, using Custom Post Types.

## Description

The **Floating Contact Button** plugin allows you to easily add a floating expandable button to your WordPress website. You can configure multiple contact methods for your visitors to reach out to you, including social media links, email, phone number, and location maps. The plugin provides a user-friendly interface in the WordPress admin dashboard to manage multiple buttons, customize their appearance, and control their visibility across different devices.

## Features

- **Multiple Links**: Add unlimited links for WhatsApp, Facebook, Instagram, LinkedIn, Twitter, Viber, Email, Phone, Maps, and more.
- **Custom Post Type**: Manage your floating buttons easily via a dedicated "Floating Buttons" menu in your WordPress admin.
- **Highly Customizable**:
  - Choose main button icon and color.
  - Set specific colors and icons for each contact link.
  - Add optional tooltip text for each link.
- **Positioning & Offsets**: 
  - Position the button at the bottom-right, bottom-left, top-right, or top-left.
  - Fine-tune positioning with custom top, bottom, left, and side offsets.
- **Size Options**: Select from predefined sizes or set custom pixel dimensions for both the main button and the individual link items.
- **Device Visibility**: Easily hide specific buttons on Desktop, Tablet, or Mobile devices.
- **Multiple Buttons**: Create and display multiple floating buttons on your website simultaneously.
- **FontAwesome Integration**: Includes FontAwesome 6.4.0 for a wide variety of high-quality icons.

## Installation

1. Upload the `floating-contact-button` folder to the `/wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Floating Buttons** -> **Add New** in the WordPress admin menu to create your first button.

## Usage

1. Go to **Floating Buttons** in the WordPress admin sidebar.
2. Click **Add New Button**.
3. **Title**: Give your button a title (for internal reference).
4. **General Settings**: 
   - Choose the main icon.
   - Select the main button color.
   - Choose the position on the screen.
5. **Appearance Settings**:
   - Select the button size or set custom dimensions.
   - Adjust the offset values to position it exactly where you want it.
6. **Visibility Settings**:
   - Choose whether to hide the button on Desktop, Tablet, or Mobile.
7. **Button Links**:
   - Click "Add New Link" to add a contact method.
   - Search or select a platform (e.g., WhatsApp, Email).
   - Enter the target URL, phone number, or map link.
   - Customize the icon, color, and tooltip text for the link.
8. Click **Publish**. The button will now be displayed on your website's front end.
